# Pharmacy Operational Dashboard Specification

## الهدف
Dashboard تشغيلية لمدير الصيدلية لعرض الأداء اليومي
بدون أي تعديل أو حذف في البيانات.

---

## 1. Dashboard Home
### يعرض:
- إجمالي مبيعات اليوم
- عدد الفواتير
- إجمالي المرتجعات
- صافي المبيعات
- رسم بياني لمبيعات آخر 7 أيام
- أعلى 5 أصناف مبيعًا
- أعلى 3 موظفين مبيعًا
- تنبيهات:
  - أصناف قربت تخلص
  - مرتجعات أعلى من الطبيعي

---

## 2. Sales Page
### Filters:
- من تاريخ → إلى تاريخ
- نوع البيع (Cash / Insurance / Delivery)
- الموظف

### KPIs:
- إجمالي المبيعات
- إجمالي الخصومات
- صافي المبيعات
- عدد الفواتير

### Table:
- رقم الفاتورة
- التاريخ
- نوع البيع
- الموظف
- الإجمالي
- الخصم
- الصافي

---

## 3. Stock Page
### يعرض:
- عدد الأصناف
- أصناف قريبة من النفاد
- أصناف راكدة

### Table:
- كود الصنف
- اسم الصنف
- الوحدة
- الكمية الحالية
- الحد الأدنى
- الحالة (آمن / تحذير / خطر)

---

## 4. Returns Page
### يعرض:
- عدد المرتجعات
- قيمة المرتجعات
- نسبة المرتجعات من المبيعات

### Table:
- رقم الفاتورة
- تاريخ المرتجع
- نوع المرتجع
- الموظف
- قيمة المرتجع
