# Pharmacy POS System

Python-based Point of Sale system designed for pharmacy operations.

## Key Features
- Cash, Insurance, and Delivery invoices
- Partial and full returns
- Arabic PDF invoice printing
- Customer & employee management
- Drug knowledge base

## Tech Stack
- Python
- Tkinter / CustomTkinter
- SQL Server
- ReportLab
- PyODBC
